Harambee Laravel Application

  • Project date: Feb, 2020 - Present
  • Project URL: davymaish/harambee
  • Tech Stacks: Laravel, MySql, HTML5, Bootstrap, jQuery

A laravel crowdfunding platform where users can create unlimited campaigns to help those in need or raise funds for community projects.

  • Designed an intuitive dashboard with features such as campaign management, media uploads and customizable user profiles.
  • Integrated payment gateways like stripe and paypal for handling one time and recurring donations, automated payouts and transaction tracking.
  • Implimented real-time updates on campaign progress through emails notifications and in-app alerts.
  • Developed a robust admin dashboard for user management, campaign moderation, analytics and content management.
